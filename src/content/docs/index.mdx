## Get started

<Aside type="tip" title="Prerequisites">
  You need an Astro project (v4 or higher) to use MMUI. If you don't have one yet, run `npm create astro@latest`.
</Aside>

Install the package:

```bash
npm install @milanlaukens/mmui
```

Import any component you need:

```astro
---
import { Nav, Hero, ProjectGrid } from '@milanlaukens/mmui';
---

<Nav align="left" background="#0f0f0f" />
<Hero />
<ProjectGrid />
```

That's it — no config files, no providers, no wrappers.

## Philosophy

<Aside type="note">
  MMUI follows a two-tier component model — **Primitives** and **Blocks** — instead of full Atomic Design. No pages, no templates. Those belong in your project, not the library.
</Aside>

The goal is a library that feels like a toolbox, not a framework. You import exactly what you need, customize it through props, and own the layout entirely. Nothing is magic, nothing is hidden.
